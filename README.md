# Mastodon Menu Bar Client

A lightweight, native macOS menu bar application for Mastodon, built with **SwiftUI**.

## Features
- 🚀 **Native Performance:** Built with SwiftUI for minimal RAM and CPU usage.
- ⚡️ **Quick Access:** Lives in your menu bar for instant "tooting" and feed browsing.
- 🖼️ **Media Support:** Upload pictures and text directly from the popover.
- 🔐 **Secure OAuth:** Standard login flow compatible with any Mastodon instance.


## Why?
Built as a Proof of Concept (PoC) to explore macOS development and the Mastodon API without the overhead of Electron-based apps.
